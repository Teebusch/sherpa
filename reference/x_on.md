# Event Listener

Listen for browser events and execute JavaScript code.

## Usage

``` r
x_on(event, code, modifiers = NULL)
```

## Arguments

- event:

  The event name (e.g., "click", "keyup", "submit").

- code:

  JavaScript code to execute.

- modifiers:

  Optional modifiers like "prevent", "stop", or "outside".
