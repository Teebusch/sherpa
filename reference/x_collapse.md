# Alpine.js Collapse Directive

Expand and collapse elements using smooth animations. Requires Plugin
"Collapse"

## Usage

``` r
x_collapse(duration = NULL, min = NULL)
```

## Arguments

- duration:

  Optional duration (e.g., "500ms")

- min:

  Optional minimum height (e.g., "50px")
