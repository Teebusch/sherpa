# Loop Through Data (x-for)

Elements inside an x-for must be wrapped in a tag.

## Usage

``` r
x_for(iteration, key_prop = NULL)
```

## Arguments

- iteration:

  Loop syntax, e.g., "item in items".

- key_prop:

  Optional property name to use as a unique key.
