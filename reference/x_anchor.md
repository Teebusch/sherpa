# Alpine.js Anchor Directive

Position an element relative to another "reference" element. Requires
Plugin "Anchor"

## Usage

``` r
x_anchor(ref, pivot = NULL, offset = NULL)
```

## Arguments

- ref:

  The reference element (e.g., "\$refs.button")

- pivot:

  The anchor point (e.g., "bottom-start", "top-end")

- offset:

  Offset in pixels
