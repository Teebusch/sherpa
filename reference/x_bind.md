# Attribute Binding

Bind HTML attributes based on JavaScript expressions.

## Usage

``` r
x_bind(attr, expression, modifiers = NULL)
```

## Arguments

- attr:

  The HTML attribute to bind (e.g., "class", "disabled", "href").

- expression:

  JavaScript expression to evaluate.

- modifiers:

  Optional modifiers, such as "camel" for SVG attributes.
