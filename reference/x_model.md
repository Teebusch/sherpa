# Two-Way Data Binding

Bind the value of an input element to Alpine data.

## Usage

``` r
x_model(prop, modifiers = NULL)
```

## Arguments

- prop:

  The property in the Alpine data object.

- modifiers:

  Modifiers like "debounce.500ms", "number", or "lazy".
